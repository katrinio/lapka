# Controls

## Global
- `/` - focus the terminal input
- `Escape` - close open suggestions and clear selected row
- `ArrowUp` - move the selected row up
- `ArrowDown` - move the selected row down
- `Enter` - open the selected row when nothing is being edited

## Terminal
- `Enter` - run the command
- `Tab` - accept the current suggestion
- `ArrowUp` - move to the previous suggestion
- `ArrowDown` - move to the next suggestion
- `Escape` - close suggestions and blur the terminal input

## Forms
- `/` - type a slash as normal text
- `Escape` - keep editing the current field
